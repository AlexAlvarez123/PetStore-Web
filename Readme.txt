* Manual de ejecución *

1. Ubicar la clase RunnerTest dentro de la carpeta src/test/java/com.bdd/runner
2. Asegurarse que el tag que se va ejecutar sea @FlujoDeCompra
3. Hacer click derecho en la clase RunnerTest y presionar 'Run RunnerTest'
4. Luego de ejecutar en Runner se podrá observar el reporte en la carpeta target/site/serenity 0a8e9939482cfbe1b9ec5d2dc7b40af1b5bbbb3deefaf1a44310ade746f0f8f6.html
